## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* node package manager ‚Äì to be able to install packages locally.
* A github Account ‚Äì to be able to use github actions.


### Setup
* Open the terminal
* Run ```cd \path\to\where\you\want\your\local\copy```
* Clone the repo
  * Run ```git clone https://github.com/Trictonicmp/bookstore-react.git```
  * Run ```cd bookstore-react```
* Install dependencies  
  * Run ```npm install```

### Build
Run ```npm run build```

### Tests
Run ```npm test```
